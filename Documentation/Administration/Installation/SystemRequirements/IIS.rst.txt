.. _system-requirements-iis:

* During the initial installation of TYPO3, the default IIS web config file is
  copied to the installation root folder.

* Default IIS web config file with rewrite rules can be found in
  :file:`EXT:install/Resources/Private/FolderStructureTemplateFiles/root-web-config`

* The `URL Rewrite plugin <https://www.iis.net/downloads/microsoft/url-rewrite>`__ is required.
