=============================
AfterFileMetaDataCreatedEvent
=============================

The PSR-14 event
:php:`\TYPO3\CMS\Core\Resource\Event\AfterFileMetaDataCreatedEvent`
is fired once metadata of a file was added to the database,
so it can be enriched with more information.
