=============================
AfterFileMetaDataUpdatedEvent
=============================

The PSR-14 event
:php:`\TYPO3\CMS\Core\Resource\Event\AfterFileMetaDataUpdatedEvent`
is fired once metadata of a file was updated, in order to update custom metadata
fields accordingly.
