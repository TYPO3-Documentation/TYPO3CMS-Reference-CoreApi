    filtered backend users.

    Listeners can modify query constraints based on currently active filtering.

    :param $demand: An instance of :php:`\TYPO3\CMS\Beuser\Domain\Model\Demand` containing the current search criteria

    :param $query: The :php:`\TYPO3\CMS\Extbase\Persistence\QueryInterface` instance being used to assemble the query
