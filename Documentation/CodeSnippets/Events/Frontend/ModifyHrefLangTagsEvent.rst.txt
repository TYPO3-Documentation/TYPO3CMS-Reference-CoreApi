..  php:class:: ModifyHrefLangTagsEvent

    Listeners to this event will be able to modify the hreflang tags that will be generated. You can use this when you
    have an edge case language scenario and need to alter the default hreflang tags.

    ..  php:method:: getHrefLangs()
        :returns: `array`

    ..  php:method:: getRequest()
        :returns: `\Psr\Http\Message\ServerRequestInterface`

    ..  php:method:: setHrefLangs(array $hrefLangs)

        Set the hreflangs. This should be an array in format:

        ..  code-block:: php

            [
                'en-US' => 'https://example.com',
                'nl-NL' => 'https://example.com/nl'
            ]

        :param $hrefLangs: the hrefLangs

    ..  php:method:: addHrefLang(string $languageCode, string $url)

        Add a hreflang tag to the current list of hreflang tags

        :param $languageCode: The language of the hreflang tag you would like to add. For example: nl-NL
        :param $url: The URL of the translation. For example: https://example.com/nl
