..  php:class:: AfterFileCommandProcessedEvent

    Event that is triggered after a file command has been processed. Can be used
    to perform additional tasks for specific commands. For example, trigger a
    custom indexer after a file has been uploaded.

    ..  php:method:: getCommand()
        :returns: `array<string,array<string,mixed>>`

        A single command, e.g.

        ..  code-block:: php

            'upload' => [
                'target' => '1:/some/folder/'
                'data' => '1'
            ]

        :returns: `mixed`

        :Return description: The result - Depending on the performed action,this could e.g. be a File or just a boolean.
