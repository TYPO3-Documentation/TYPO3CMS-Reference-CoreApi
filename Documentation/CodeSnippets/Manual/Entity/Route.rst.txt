..  php:class:: Route

    This is a single entity for a Route.

    The architecture is highly inspired by the Symfony Routing Component.

    ..  php:method:: getPath()
        :returns: `string`

        Returns the path

        :Return description: The path pattern

    ..  php:method:: setPath(?string $pattern)
        :returns: `\Route`

        Sets the pattern for the path
        A pattern must start with a slash and must not have multiple slashes at the beginning because the
        generated path for this route would be confused with a network path, e.g. '//domain.com/path'.

        This method implements a fluent interface.

        :param $pattern: The path pattern
        :Return description: The current Route instance

    ..  php:method:: getMethods()
        :returns: `string[]`

        Returns the uppercased HTTP methods this route is restricted to.

        An empty array means that any method is allowed.

        :Return description: The methods

    ..  php:method:: setMethods(array $methods)
        :returns: `self`

        Sets the HTTP methods (e.g. ['POST']) this route is restricted to.

        An empty array means that any method is allowed.

        This method implements a fluent interface.

        :param $methods: The array of allowed methods
